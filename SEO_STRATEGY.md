# CantoSub AI - SEO 推廣策略指南

**目標市場：** 香港、台灣、馬來西亞、新加坡等粵語/中文內容創作者

**核心關鍵詞：**
- 主要：AI字幕、自動字幕、廣東話字幕、影片字幕生成
- 次要：字幕製作、語音轉文字、OCR字幕、YouTube字幕

## 一、技術 SEO

### 1.1 網站結構優化

```
首頁 (/) - 核心關鍵詞：AI字幕生成器
├── 功能 (/features) - 功能介紹
├── 定價 (/pricing) - 價格頁
├── 關於我們 (/about) - 品牌故事
├── 部落格 (/blog) - SEO內容
│   ├── 如何為YouTube影片上字幕

│   ├── 字幕轉換器
│   └── 字幕編輯器
```

### 1.4 性能優化

- **Core Web Vitals**
  - LCP (Largest Contentful Paint) < 2.5s
  - FID (First Input Delay) < 100ms
  - CLS (Cumulative Layout Shift) < 0.1

- **圖片優化**
  - WebP 格式
  - 延遲加載 (lazy loading)
  - 響應式圖片

- **代碼分割**
  - 路由級代碼分割
  - 組件延遲加載

## 總結

SEO 是長期投資，預期 3-6 個月開始看到效果。重點：

1. **技術基礎** - 確保網站可被搜尋引擎正確抓取
2. **優質內容** - 解決用戶問題的內容
3. **持續更新** - 定期發布新內容
4. **外鏈建設** - 獲取高質量反向連結
5. **數據驅動** - 根據數據調整策略
